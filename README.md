# Cooling Control System Simulation

This project simulates a cooling control system for an inverter and DC-DC platform.

## Features
- State machine (OFF / NORMAL / OVERHEAT / FAULT)
- Temperature-based control logic
- Simulated CAN communication
- Modular C++ design
- CMake build system

## Build & Run

```bash
chmod +x scripts/run.sh
./scripts/run.sh
